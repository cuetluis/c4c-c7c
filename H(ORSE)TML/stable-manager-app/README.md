# Stable Manager App

## Overview
The Stable Manager App is a mobile application designed for managing horse treatments, daily observations, and audit trails. It provides a user-friendly interface for tracking horses, treatments, actions taken, and daily tasks.

## Features
- **Horse Table**: View and select horses for treatments.
- **Treatment Table**: Display available treatments and allow multi-selection for horses.
- **Action Taken Table**: Log actions taken on treatments, including notes and timestamps.
- **Daily Observations/To-Do List**: Track daily observations and tasks for staff.
- **Audit Trail**: Log changes made to any table, detailing who made the change and when.

## Installation
1. Clone the repository:
   ```
   git clone https://github.com/yourusername/stable-manager-app.git
   ```
2. Navigate to the project directory:
   ```
   cd stable-manager-app
   ```
3. Install dependencies:
   ```
   npm install
   ```

## Running the App
To run the app on iOS:
```
npx react-native run-ios
```

To run the app on Android:
```
npx react-native run-android
```
